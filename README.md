# 🏠 Rosam Pisos - Catálogo Online

Catálogo completo de pisos e revestimentos de alta qualidade da **Rosam Pisos**.

## 📋 Características

- ✅ **136 produtos** com imagens e descrições
- 🔍 **Busca em tempo real** por nome, tamanho ou tipo
- 🎨 **8 categorias** de pisos (Azul, Cinza, Claro, Dourado, Madeira, Marmorizado, Outros, Preto)
- 📱 **Design responsivo** - funciona em celular, tablet e desktop
- 💬 **Integração com WhatsApp** - botão direto para contato
- ⚡ **Sem necessidade de login** - acesso público e gratuito

## 🚀 Como Usar

1. Abra o arquivo `index.html` no seu navegador
2. Use a barra de busca para encontrar produtos
3. Clique nos filtros de categoria para visualizar por tipo
4. Clique em "Solicitar Orçamento" para enviar mensagem via WhatsApp
